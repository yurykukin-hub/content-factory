# НаWоде — промо-плейбук (акции, конкурсы, UGC) + экономика

> Создан 2026-06-22. Источник: разбор biz-strategist (opus) + marketing-content (sonnet) + реальные данные ERP НаWоде (цены, оплата инструктора `compensation_rules`, скидочные механизмы). Зафиксировано, чтобы механики не «уплыли» и был порядок тестов. Связано с планом `~/.claude/plans/robust-scribbling-harp.md` и фазами Content Factory.

## Принцип №1 (на нём держится вся экономика)
Место в туре/слоте — **скоропортящийся товар**: пустое место = 100% упущенная выручка, переменные затраты на +1 участника ≈ 0 (доска есть, инструктор уже едет). Значит:
- **Скидка ДОБИВАЕТ то, что и так едет** (полупустой идущий слот) → почти чистая маржа.
- **Скидка НЕ должна плодить новые слоты** под скидочников (аренда + время инструктора → убыток).
- **Пик не трогаем скидками** (выходные, закаты, июль-август — продаётся и так).

## 1. Экономика и предельные скидки (по ERP `compensation_rules`)

**Оплата инструктора (факт из ERP):**
- Смена/выход (прокат+уроки): `max(фикс-мин, Σбонусов)`. Короткий выход ~3-4ч: фикс-мин 500₽ (прокат +100₽/шт, урок 1ч +200₽, 1.5ч +250₽). Полная смена ~8ч: фикс-мин 2000₽.
- Туры Монрепо/Беличьи/Место силы (4000₽/чел): на ГРУППУ ступенями — 2-3 чел = 2500₽, 4-7 = 3000₽, 8+ = 3500₽.
- Закат: фикс 2000₽ на любую группу. На скале: прогрессивно 2000 / +500 / +1000 / +1500.
- Аренда набережной = 10% оборота (на Дружбе фикс — там скидки ещё безопаснее).

**Маржа на ДОП. участника идущего слота (нижняя граница):**

| Услуга | Маржа +1 чел |
|---|---|
| Прокат Выборг (1000₽) | +800₽ |
| Прокат 3.5ч (2000₽) | +1700₽ |
| Прогулка «Выборг с воды» (1500₽) | +1250₽ |
| **Закат (2500₽)** | **+2250₽** (инструктор фикс → каждый сверх 1-го почти чистыми) |
| **Тур (4000₽)** | **+3600₽ внутри ступени / +3100₽ на переходе** (4-й и 8-й чел) |
| Арома-завтрак (2500₽) | ~+1900₽ (минус себестоимость еды ~350₽/чел — единственная услуга с товарной частью) |

**Точка безубыточности:** прокат короткий выход — 1 чел; полная смена 8ч — 3 чел; закат — 1 чел; тур — 1 чел уже в плюс (обычно от 2). **Туры добивать 3→4 и 7→8 ВСЕГДА выгодно** (шаг ступени 500₽ ≪ маржа 3600₽ — ловушки нет).

**Красные линии (предельная скидка):**
- **Добивка (один лишний в идущий слот):** глубоко — прокат до −75% (мин ~250₽), закат до −88%, тур внутри ступени до −87%. НО завтрак только до −72% (еда), тур на переходе ступени до −75% (~1000₽).
- **«Второй за 1₽»:** безопасно ТОЛЬКО на второго при платном первом (пара проката = +601₽). «Оба за 1₽» — убыток.
- **Массовая акция (на всех): безопасный потолок −40%** по всему меню (везде ≥400₽/чел). Будни вт/пт можно углублять с текущих −10% до −30-40% (риск поведенческий — приучить ждать, не финансовый).
- **НИКОГДА:** тур глубже −75% на переходе ступени; завтрак глубже −70%; «весь слот по линии выживания» массово (обнуляет цену в голове рынка).

**Открытый вход:** реальная себестоимость продуктов на Арома-завтрак (заложено ~350₽/чел) — влияет ТОЛЬКО на линию завтрака.

## 2. «Прокат/услуга за 1₽» (порядок тестов)
1. **① Второй за 1₽ (реферал) [тест первым].** Платный гость (1000₽) → промокод «друг за 1₽» на тот же слот (лимит 1, привязка к дате, не суммируется со скидочными днями). Апселл пары, заполняет оплаченный слот, CAC≈0.
2. **② Утренний якорь за 1₽ [тест вторым, синергия со слот-филлом].** 1 место за 1₽ в мёртвый будний утренний слот; якорь запускает добивку через social proof. Строго 1 место/слот, бронь заранее, мин. N для выхода.
3. **③ Секретный промокод недели.** −30-40% на непиковый день, лимит использований = вместимость 1 слота (потолок убытка ≈ 5×400₽). FOMO + ежедневный заход в профиль. Ротировать день.
- ❌ «−99% всем раз в неделю» — НЕ делать (убивает средний чек, приучает ждать).

## 3. UGC-бартер (отдаём только пустые места → себестоимость ~0)
- **Микро (1-10к, локальные Выборг/СПб):** прокат/прогулка на свободное место идущего слота. Требуем: 1 пост ИЛИ 2-3 сторис + гео + @nawode + право на репост, срок 3 дня. Лимит 1-2/нед, только непик.
- **Средние (10-50к, тревел/лайфстайл СПб):** место в группе где уже есть платящие ИЛИ −50% (не бесплатно — «кожа в игре»). Требуем Reels (×2 охвата) + сторис + репост-права. Отбор по просмотрам рилсов, не подписчикам.
- **Правила:** ≤10-15% мест за сезон под бартер; условия письменно ДО; реестр (кому/что получили/охват/брони); невыполнил → чёрный список.

## 4. Календарь акций (механизмы ERP уже есть)
- **Дни недели** (`price_lists.discount_days`): будни −15-20% прокат (вт/пт уже 900₽). Выходные — без скидок, только апселл.
- **Время:** утро будни — добивка (раздел 2②); день будни «обеденный SUP» −15%; закат/Белые ночи — премиум, слот-филл «места тают».
- **Сезон:** май — «открытие −20%» + сбор UGC; июнь-авг — точечная добивка, флагманы по полной; сентябрь — «золотая осень −20%» + подарочные сертификаты.
- **Погода (flash):** распогодилось → −30% на сегодня (разовая скидка `service_slot_overrides`). Кандидат №1 на автоматизацию (CF Фаза 3).
- **Добивка слотов:** за N часов до слота 2-3/9 → «−25% последним». +3 чел × 750₽ ≈ +2250₽ почти чистыми.

## 5. Конкурсы (приз = пустое место / отложенная скидка → окупаемы)
- **«Кадр недели»** (#НаВоде_Выборг): репост лучшего раз в неделю (пт), приз = статус/место. UGC-машина, лучший ROI, решает дефицит визуала.
- **«Отметь друга — прогулка на двоих»** (репост+отметка): охват, подписчики, призёры часто возвращаются платно.
- **«Угадай место» по фото маршрута:** комментарии (алгоритм VK любит) + прогрев к турам. Серия из банка гео-фото.
- **«Сертификат за отзыв»** (VK/Яндекс/2ГИС → −20% на следующую): отзывы = долгий рост, скидка списывается только при след. покупке.
- **«Пиратская гавань 2026»** (сезонный пик): розыгрыш участия/мерча как инфоповод за месяц.

## 6. Серийные рубрики (тренд 2026: сериальность > виральность)
- Сериалы из 10 рубрик: «Маршруты с воды» (VK Клипы, арка май→сентябрь), «От нуля до уверенного гребка» (новички), «Команда НаWоде» (портреты). Каждый выпуск тизерит следующий.
- **Структура разбора чужого рилса** (доска вдохновения, CF Фаза 4): Хук (первые 3 сек: тип+что) · Структура (хронометраж/формат/темп) · Ключевая идея+эмоция · Применимо для НаWоде (хук-адаптация/воспроизводимо да-нет/в какую рубрику) · Что НЕ повторять · Готовый промпт для CF.

## 7. Анти-риски / РФ
- Не делать скидку на один день недели постоянно (выучат) — ротировать; пик держать без скидок; мерить средний чек до/после.
- **Маркировка рекламы** у блогеров (бартер = реклама: пометка + ОРД/erid). **Instagram** — только органика, без рекламы, не афишировать в VK/TG (Meta экстремист в РФ). **Налоги/чек:** бартер = реализация по рыночной стоимости (УСН), «1₽» — чек на 1₽ ок, систематика может привлечь внимание; свериться с бухгалтером. Промокоды — публиковать условия (срок/лимит/на что).

## 8. Топ-3 к внедрению первыми (быстро + прибыльно)
1. **Автодобивка полупустых слотов (слот-филл)** — №1 по деньгам, почти чистая маржа. Данные уже есть в CF (`getHotSlots`, прод 2026-06-22). Дневная сторис ритма уже промоутит горячий слот.
2. **Погодный flash-промо** — почти нулевой риск, погода уже читается. CF Фаза 3.
3. **UGC «Кадр недели» + микро-бартер** — призы ~0₽, решает дефицит визуала, растит охват.
